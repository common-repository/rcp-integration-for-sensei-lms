Sell online courses with Sensei LMS & Restrict Content Pro

== Description ==
With this integration you will be able to create membership levels in **Restrict Content Pro** and then associate them to **Sensei LMS** courses.

Customers will be auto-enrolled into courses after signing-up a membership level.

= Features =

= #1. Associate membership levels to one or more courses: =

Create a membership level and associate it to one, or multiple, Sensei LMS courses. 

So, you can sell courses separately, by bundles or by subscription.

= #2. Automatic removal upon membership cancellation: =

When a user cancels their membership then the plugin will automatically unenroll them from their courses as well.

*It takes into account whether a course was previously sold separately with a lifetime access.*

== Installation ==
= Automatic Installation =

1. Log into your WordPress admin panel and go to *Plugins > Add New*.
2. Enter “RCP Integration for Sensei LMS” into the search field.
3. Once you’ve located the plugin, click Install Now.
4. Click Activate.
5. Once the plugin is installed and active, you simply need to create or edit a membership level in *Restrict > Membership Levels > Courses* and associate it to one, or multiple, courses.

== Frequently Asked Questions ==
= Which plugins do I need to have activated? =

To be able to run **RCP Integration for Sensei LMS** you must have activated the plugins **Sensei LMS** and **Restrict Content Pro** (*version 3.0 or higher*).

= Where is the plugin configuration? =

In *Restrict > Membership Levels* when you add or edit an existing one.

= Can I associate a free membership level to courses? =

Yes, you can do it.

= What happens if a customer cancels a recurring subscription that includes one (or multiple courses) that has previously bought a lifetime access? =

The customer will maintain the access and progress of these courses.
